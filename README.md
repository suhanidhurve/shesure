# SheSure 💗

## Know What You Wear

SheSure is a menstrual health awareness website that helps users understand different menstrual products like pads, tampons, menstrual cups, and period panties.

The website provides:

- Product safety information
- Safety quizzes
- Hygiene tips
- Personalized recommendations
- FAQ section
- Feedback form

## Features

- Interactive user interface
- Menstrual product awareness
- Safety score system
- Product recommendation system
- Educational content
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- FormSubmit

## Purpose

The purpose of this project is to spread awareness about menstrual hygiene and help users make safe and informed choices without promoting any specific brand.
